Adaptive AI Context Interface for MX Creative Console

DialMind maps physical dials/buttons/LED feedback to live IDE context so developers can interact with AI suggestions and debug flows without breaking keyboard focus.

Toggle AI-assist session mode

State awareness: ready, paused, debug urgency
